Robotics and Automation

I've collaborated in a couple of projects with training partners such as ABB, Omron Automation, John Deere, and Acuity Brands

Omron Project (Ciber-Physical systems I)

The main objective proposed by training partner Omron Automation is to integrate their equipment, a TM5M-700 manipulator mounted on an AGV to perform supply and removal of material (micro-logistics) from the Raw Material warehouse to the conveyor belts and Finished Product warehouse. My main involvement in this process is to perform collaborative robot programming of the TM5M-700, creating Pick and Place operations for the removal and manipulation of material, utilizing its 5M pixel integrated camera for image processing, QR Scanning for traceability and robotic visual servoing, color detection for quality control and interfacing using Modbus TCP/IP communication.

ABB Welding Cell

Using CAD tools like SolidWorks, this project involved the design of an automated welding station using an ABB industrial robot, adhering to national and international standards such as: NOM-004-STPS-1999, NOM-018-STPS-2000, NOM-026-STPS-1998 and the ANSI RIA R15.06-1999 to verify the compliance of the proposal with current industry regulations. Validation was done using ABB RobotStudio and Cognex Insight Explorer for the integration of industrial vision systems.

Parker Hannifin, Sporlan Division

In this project, a proposal was developed to automate the manufacturing process of thermo-expansive valves from a local factory, the end result provided a solution using UR collaborative robots that could outperform tasks manually done by operators by reducing cycle times. The whole process was recreated using Tecnomatix Process Simulate

John Deere
Autonomous Vehicle

In this project, we designed an autonomous vehicle using an STM32 microcontroller The main focus this project was the integration of multiple sensor input and the development of PID control systems to achieve the desired trajectory to provide a higher level of autonomy for the prototype, recreating the autonomous tractor presented at CES 2022. For this particular project my main task involved programming in C using STM32CubeIDE, electronic circuit design for the vehicle, PCB design and layout using KiCad, and implementing control systems for speed and orientation.

Electronic Speed Controller

For the debut of Tecnológico de Monterrey Monterrey Campus (EcoShell Racing) at the Shell Eco-marathon Americas 2024, as the electronics team leader I developed a motor controller for a 500 W BLDC motor.
